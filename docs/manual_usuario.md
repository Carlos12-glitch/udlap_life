1. Introducción

UdlapLife es una aplicación web diseñada para la comunidad UDLAP con el objetivo de centralizar servicios académicos, administrativos e informativos en una plataforma moderna, rápida e intuitiva.

Mediante esta aplicación, los usuarios pueden consultar información relevante de manera sencilla, mejorando la experiencia respecto a plataformas anteriores.

2. Acceso al Sistema

Paso 1: Abrir la aplicación

Ingresar desde el navegador web a la dirección proporcionada por el equipo de desarrollo.

Ejemplo:

http://localhost:5173/
Paso 2: Iniciar sesión

En la pantalla principal, capturar:
- ID institucional
- Contraseña

Presionar el botón Iniciar Sesión.

Nota
El sistema valida las credenciales institucionales del usuario.

3. Pantalla Principal

Después de iniciar sesión, el usuario accede al menú principal, desde donde puede navegar entre los diferentes módulos del sistema.

Funciones principales disponibles:

- Horario de clases
- Estado de cuenta
- Transporte UDLAP
- Mapa del campus
- Menú de cafeterías
- Transcript académico
- Calificaciones
- Configuración

4. Módulos del Sistema

4.1 Horario de Clases

Permite consultar el horario actual de materias inscritas.

Funciones:
- Visualizar materias
- Consultar horarios
- Descargar horario

4.2 Estado de Cuenta

Muestra información financiera del usuario.

Funciones:
- Consultar saldo pendiente
- Revisar fechas límite de pago
- Visualizar cargos realizados
- 4.3 Transporte UDLAP

Presenta información relacionada con rutas y paradas de transporte institucional.

Funciones:
- Ver horarios
- Consultar rutas disponibles
- Ubicar paradas

4.4 Mapa del Campus

Permite localizar edificios y puntos importantes dentro del campus.

Funciones:
- Navegación visual
- Ubicación de instalaciones
- Consulta rápida de áreas

4.5 Menú de Cafeterías

Muestra alimentos disponibles y menú del día.

Funciones:
- Consultar cafeterías
- Ver menú actualizado
- Revisar opciones disponibles

4.6 Transcript Académico

Permite visualizar historial académico.

Funciones:
- Consultar materias cursadas
- Ver promedio
- Descargar transcript

4.7 Calificaciones

Muestra las evaluaciones y calificaciones obtenidas por el usuario.

Funciones:
- Ver materias
- Consultar notas finales
- Seguimiento académico

4.8 Configuración

Permite personalizar opciones del sistema.

Funciones:
- Ajustes visuales
- Preferencias de usuario
- Opciones generales

5. Navegación del Sistema

La aplicación cuenta con una interfaz intuitiva que permite desplazarse fácilmente entre módulos mediante menús y botones de acceso rápido.

6. Recomendaciones de Uso

- Utilizar navegador actualizado.
- Mantener conexión a internet estable.
- Cerrar sesión al terminar.
- No compartir credenciales institucionales.

7. Solución de Problemas
No puedo iniciar sesión

Verificar ID y contraseña.

La página no carga

Revisar conexión a internet o recargar navegador.

Información desactualizada

Cerrar sesión e ingresar nuevamente.

8. Cierre de Sesión

Para salir del sistema, utilizar la opción Cerrar Sesión dentro del menú principal o configuración.

9. Soporte

Para reportar errores o solicitar mejoras, contactar al equipo desarrollador de UdlapLife.
