📘 UdlapLife – Repositorio de Documentación

Bienvenido al repositorio de documentación del proyecto UdlapLife, una aplicación web desarrollada para mejorar la experiencia digital de la comunidad UDLAP mediante una plataforma moderna, rápida e intuitiva.

Este directorio concentra los documentos técnicos y de usuario relacionados con la instalación, uso y mantenimiento del sistema.
